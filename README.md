# ShoeTracker
mileage tracker for my running shoes, made to understand C# basics

Structure is as follows.

```text
ShoeTracker/
├── ShoeTracker.csproj
├── Program.cs               < interactive menu (entry point)
├── Models/
│   ├── Shoe.cs              < Shoe class
│   └── Run.cs               < Run class + RunType enum
└── Services/
    └── TrackerService.cs    < business logic + LINQ query
```
